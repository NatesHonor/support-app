Installation

Option 1 - Using the CLI

The following steps explains how to install the Missionchief Bot using the Command Line Interface (CLI)

Step 1 - Downloading the Bot

  1. First step is to of course download the bot

  2. When on the download page you can either Download the Latest Version or scroll and download a previous version.

  3. After finding the version you want it should navigate you to the correct download link.

All you have to do is wait 5 seconds and the file will be downloaded automatically!

Step 2 - Extracting the bot

All you have to do is find the downloaded zip file, and extract it to a folder of your choice!

Step 3 (Optional) - Creating the venv!

  1. First, you want to open the CLI of your choice, for this example I use Command Prompt

  2. Run the command python -m venv .venv

  3. Then you want to activate the venv using .venv\Scripts\activate

Step 4 - Installing the requirements

If you created a venv make sure it's activated

All you have to do is run the command pip install -r requirements.txt

Step 5 - Configuring the bot

First thing first, you want to open the config.ini and edit the line username and password and set it to your email/username and password.

Step 6 - Running the bot

All you have to do is run python main.py and if you followed the steps correctly it should start running!

Option 2 - Using the Missionchief Bot.exe

Step 1 - Downloading the Bot

  1. First step is to of course download the bot

  2. When on the download page you can either Download the Latest Version or scroll and download a previous version.

  3. After finding the version you want it should navigate you to the correct download link.

All you have to do is wait 5 seconds and the file will be downloaded automatically!

Step 2 - Extracting the bot

All you have to do is find the downloaded zip file, and extract it to a folder of your choice!

Step 3 - Run the exe file

  1. First double-click the bot, no administrator access is required.

  2. Next Click the "edit config.ini" button to customize the config

  3. Now just click the "start bot" and the exe will handle the rest!
